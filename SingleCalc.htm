Single Stage Distortion Calculator

Pls note: The Bipolar/JFET switch is not implemented yet. ActiveX control is 170 kB, no dependencies. Sorry about the time it took to finish this, but this site is only updated on a very unregular schedule :-)

The single common emitter stage is modelled above (and shown to the right) as an ac/signal model.  The distortion is calculated from the base-emitter nonlinearity. Only the Current (Emitter current), emitter resistor (which may be zero) and input voltage are required for calculating the distortion.

The Ucc and the Rc are calculated assumed the DC model is equal to the ac model. The Rc is calculated for maximum output voltage excursion.
